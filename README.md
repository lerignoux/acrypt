lerignoux-acrypt
==============
personal extension of the ccrypt project.

Creation instructions
--------------

build package:
- dpkg --build acrypt/ acrypt-1.0_all.deb
- install sudo dpkg -i acrypt-1.0_all.deb

C++ project Dev setup:
--------------
- sudo apt-get install g++
- sudo apt-get install qt4-dev-tools

- qmake -project
- qmake

- make
